### Final Colour Exercise

A simple tool that allows the user to draw on the canvas. Colour interpolation was used to lerp between various hues. The R key can also be pressed to draw a square at the mouse's current position. If this key is pressed while the mouse is held, the square will be the same colour as the circular "paintbrush".

### Final Shape & Pattern exercise - Wrapping Paper project

For this project, the task was to create a pattern that could be used as Christmas wrapping paper for a bar of chocolate. I researched various different patterns for inspiration before deciding on a zig-zag pattern. The inspiration images and images of the finished pattern can be found here: https://projects.invisionapp.com/d/main?origin=v7#/projects/boards/6767289

This project underwent several iterations, first to test the design and colours and second, to create the full size pattern.
